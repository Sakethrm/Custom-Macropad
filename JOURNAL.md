  ## Day 1📅  
### **Time spent: 1hr 25min**🕒  


---------------------------------------------------------
Completed the schematic of my custom macropad. Added six 'Sw_push' switches for 5 keys and one mode switch!  

	  
Mode switch lets us change between c++ programming mode and everyday shortcut mode.🌟  


		
I also added 6 neopixels for using them as backlight for the keys and the mode switch!💡  


		
Rotary encoder switch can be used for adjusting volume, scrolling or zooming.🖱️
